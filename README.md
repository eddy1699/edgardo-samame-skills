# 🧠 edgardo-samame-skills

Colección de skills para instrucciones estructuradas que convierten a la IA en un especialista para tareas reales.

Cada skill define cuándo activarse, cómo trabajar paso a paso y qué entregar. Están diseñados para ser usados directamente, adaptados a tu contexto o tomados como base para crear los tuyos.

## ¿Para quién es esto?

Para cualquier persona que use cualquier IA: devs, product managers, consultores, educadores o equipos de negocio. No se necesita saber programar para usarlos.

## ¿Cómo usar un skill?

1. Entra a la carpeta del skill que necesitas
2. Copia el contenido del archivo `SKILL.md`
3. Pégalo como instrucción en Claude (Projects > Instructions)
4. Describe tu caso y deja que el skill guíe el resultado
